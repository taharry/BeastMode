# BeastMode

A Flutter fitness tracking app with Firebase backend that allows users to log workouts, track progress, and stay consistent.
## Features
- 🔐 Firebase Authentication (Login / Signup)
- 📊 Workout tracking with Firestore
- 📈 Dashboard with real-time stats
- 👤 Profile with editable display name
- 🌙 Light / Dark mode toggle
- 💬 Dynamic motivation quotes
